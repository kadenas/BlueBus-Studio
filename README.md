# BlueBus Studio

BlueBus Studio es un simulador de redes náuticas enfocado en diseñar y validar circuitos NMEA-0183 y NMEA-2000 antes de la instalación a bordo. Este repositorio contiene el primer andamiaje funcional de la aplicación Flutter, con navegación básica, un tema oscuro profesional y pantallas iniciales para el flujo principal.

## Contenido
- Pantalla de bienvenida con logotipo e inicio de flujo de proyectos.
- Editor con panel de dispositivos, área de lienzo y diagnóstico preliminar.
- Modelos de datos iniciales y controlador de proyecto para futuras extensiones.

## Requisitos
- Flutter 3.0 o superior.

## Ejecución
```bash
flutter run
```
